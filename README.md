# DANES ScHack 2026 — Website

Website for the **First DANES Summer School and Hackathon (DANES ScHack 2026)**,
Palazzo Venturi, Università di Torino, 6–10 July 2026.

Live at **[digitalpasts.github.io/ScHack2026](https://digitalpasts.github.io/ScHack2026)**
(static HTML, published via GitHub Pages from the `main` branch root).

This repository will also host teaching materials from the school.

## Local preview

No build step. Open any `.html` file in a browser, or run:

```bash
python3 -m http.server 8000   # then visit http://localhost:8000
```
