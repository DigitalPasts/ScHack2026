DANES ScHack 2026 — Images directory
=====================================

Files to add:

1. danes-logo.png
   Source: Download from https://opendanes.org or the OpenDANES GitHub repo
   (https://github.com/DigitalPasts/openDANES). Used as favicon and nav logo.

2. unito-logo.png
   Source: Download the official University of Turin logo from https://www.unito.it
   Place the PNG version here. Used in the organisers row on index.html.

3. hero-bg.jpg  (optional but recommended)
   A high-quality image of Turin (e.g. Mole Antonelliana) or ancient Near Eastern
   imagery (e.g. cuneiform tablet, ziggurat). Use a CC-licensed image from
   Wikimedia Commons. Reference in styles.css hero background-image property.

All images should be web-optimised (max ~500 KB for hero, max ~100 KB for logos).
